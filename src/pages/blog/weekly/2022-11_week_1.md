# exifの削除(Linux)

[Remove Exif Data in Linux Command Line](https://linuxhandbook.com/remove-exif-data/)

```bash
$ sudo pacman -S perl-image-exiftool
$ exiftool -overwrite_original -all= <image_path>
```
